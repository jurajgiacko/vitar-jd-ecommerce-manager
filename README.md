## Custom doména (neskôr)

Ak chceme `kariera.vitar.cz`:
1. Pridať `CNAME` súbor s obsahom `kariera.vitar.cz`
2. V DNS Vitaru pridať CNAME record: `kariera` → `jurajgiacko.github.io`
3. V GitHub repo Settings → Pages → Custom domain
